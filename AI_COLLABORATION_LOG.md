# AI Collaboration Log

## AI Tool Used

* ChatGPT (OpenAI)

## How AI Was Used

AI was used as a development assistant throughout the project. Specifically, it helped with:

* Designing the overall MVVM-based project structure.
* Creating and refining Codable models for parsing the server-driven JSON schema.
* Implementing polymorphic field handling through the FieldType enum and custom decoding logic.
* Building a dynamic field renderer that maps field types to reusable SwiftUI components.
* Implementing form validation for required fields, dropdown selections, and checkbox acceptance.
* Managing dynamic form state using dictionaries keyed by field identifiers.
* Handling default values provided by the JSON configuration.
* Implementing max-length restrictions for text fields.
* Adding support for unknown field types to prevent application crashes from unsupported server data.
* Implementing theme handling using dynamic colors from the JSON response.
* Debugging SwiftUI binding issues for text fields, toggles, dropdowns, checkboxes, and color picker components.
* Reviewing payload generation logic used when submitting form values.
* Assisting with Git and GitHub repository setup, branching, merge conflict resolution, and remote repository configuration.
* Reviewing README documentation and submission materials.

## Final Responsibility

All implementation decisions, code integration, debugging verification, testing, project organization, and final submission preparation were completed by me.
